Brief

This project will be a vinyl record catalogger for my own personal use. I want to be able to capture the records in my collection, and log them into a database that I can keep track of and value my collection. I also want to use the Discogs API and other music db APIs to autofill things like album covers and album information

My audience will be anybody with a record collection.

This will be a single page app, similar to material design in look and feel. This will be a primarily mobile application, but will work on any screen size

My stretch goal would be to implement outside APIs for completing album info and getting value for each record.

What the site will include: Add new album

Name, Artist, value, date purchased...
